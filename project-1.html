Project 1 - Ethical Hacking in Juice Shop Lab

Setting up my own cybersecurity lab turned my home workspace into a playground for breaking (and fixing) things. I crafted a secure yet vulnerable environment where I could dive into penetration testing, networking, and real-world vulnerabilities without worrying about collateral damage. Each tool added a new dimension, from testing Juice Shop's defenses to analyzing network packets with Wireshark.

Intro

In this project, I focused on network analysis and simulated attacks within a controlled environment. Using VirtualBox and a set of VMs, I configured a network and conducted penetration testing exercises to identify security gaps. Here you can see me confirming VM connectivity using "ping" on each other.

I utilized Wireshark to capture and analyze packets, tracking network activity during simulated attacks. This approach gave me practical experience in interpreting network traffic and spotting potential vulnerabilities. I experimented with tools like Burp Suite for web testing and simulated brute-force attempts to see where typical defenses fall short. Below I am capturing wfuzz brute-force traffic.

This hands-on project was a deep dive into network security, reinforcing the theory through practice. By the end, I could detect and understand various attack vectors, which significantly boosted my confidence in handling real-world security assessments.
